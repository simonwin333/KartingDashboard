📄 Conditions d'utilisation

1. Utilisation

Karting Dashboard est une application personnelle de suivi de performances en karting, fournie "en l'état" pour usage personnel.

2. Données personnelles

Vos données sont stockées sur Firebase (Google). Nous ne partageons jamais vos données avec des tiers.

3. Authentification

Votre e-mail sert uniquement à l'authentification. Vous êtes responsable de la sécurité de votre compte.

4. Responsabilité

L'application ne peut être tenue responsable des décisions prises sur la base des données affichées.

5. Suppression

Vous pouvez supprimer votre compte et toutes vos données à tout moment dans les Réglages.

❓ Questions fréquentes

Comment ajouter une session ?
Appuyez sur "➕ Session", remplissez les champs et validez. La session apparaît immédiatement dans le dashboard.
Comment est calculé mon meilleur temps ?
C'est le temps minimum parmi toutes vos sessions sur un circuit. Il s'affiche en vert dans l'analyse des circuits.
Comment ajouter un circuit ?
Dans le formulaire de session, appuyez sur "➕" à côté du champ Circuit, entrez le nom et confirmez.
Puis-je utiliser l'app sur plusieurs appareils ?
Oui ! Connectez-vous avec le même compte sur tous vos appareils. Les données se synchronisent automatiquement.
Comment installer l'app sur mon smartphone ?
Réglages → "📲 Installer l'app". Sur iOS Safari : Partager → Sur l'écran d'accueil.
Comment modifier ou supprimer une session ?
Dans l'Historique, utilisez ✏️ pour modifier et 🗑️ pour supprimer.
Pourquoi enregistrer la météo ?
La météo influence les temps. En la comparant, vous analysez mieux vos vraies performances.
Mes données sont-elles sécurisées ?
Oui. Stockées sur Firebase (Google), seul votre compte y a accès.
Comment supprimer mon compte ?
Réglages → "⚠️ Supprimer mon compte". Confirmez avec votre nom. Suppression définitive.
